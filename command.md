## একনজরে পার্থক্য

| কমান্ড                       | কাজ                                                        | ব্যবহারের ক্ষেত্র                      |
| :--------------------------- | :--------------------------------------------------------- | :------------------------------------- |
| `git log --online graph`     | কমিটের সংক্ষিপ্ত বিবরণ ও গ্রাফিক্যাল রিপ্রেজেন্টেশন দেখায়। | দ্রুত কমিট ইতিহাস দেখতে চাইলে।         |
| `git log -p`                 | বিস্তারিত কোড পরিবর্তন (Diff) দেখায়।                       | গভীর বিশ্লেষণ ও কোড রিভিউ।             |
| `git log --stat`             | ফাইলের নাম ও লাইনের সংখ্যার হিসাব দেখায়।                   | দ্রুত ওভারভিউ বা সামারি দেখা।          |
| `git log -p --stat`          | একই সাথে উভয় তথ্য দেখায়।                                 | বিস্তারিত ও সামারি একসাথে দেখতে চাইলে। |
| `git config --global --list` | আপনার কম্পিউটারে কোন কোন শর্টকাট সেট করা আছে তা দেখতে      | আপনার গিট কনফিগারেশন চেক করতে।         |
